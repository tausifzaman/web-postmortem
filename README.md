## About Web-Postmortem

**Web-Postmortem** is a powerful **website intelligence & reconnaissance tool** designed to analyze web applications and expose publicly accessible resources.

It helps developers, security researchers, and pentesters understand how a website is structured by extracting:

- URLs  
- Subdomains  
- Parameters  
- Admin panels  
- Sensitive files  
- JSON / API endpoints  

 Live Tool:  
 https://tausifzaman.online/tool/web-postmortem  

![](https://i.imgur.com/waxVImv.png)

## Usage

 Open the tool  
 https://tausifzaman.online/tool/web-postmortem  

1. Paste any website link  
2. Click **Search**  
3. Review extracted data  

No installation required.
